Flights Tableau Visualisation

This Tableau Visualisation is compiled from self scraped and cleansed UK flight data spanning 2019-2021. The visualisation compares airport and flight activity year on year in this time period. Using the dashboard you are able compare airport performance before and during covid. The dashboard demonstrates common analysis of time series data and packages it into a presentable, interactive Tableau format.

Portfolio Analysis and Prediction

Given a history of stock purchases, a portfolio is automatically generated. By scraping online data, correlation and risk assessment can be determined for the specific portfolio, while also calculating current and future ROI. Further visualisations can also be made for the portfolio.

Developed a data pipeline using the alpha-vantage API to access minute-by-minute stock data and package it into a Pandas dataframe. These large datasets are then used in conjunction with a windowed-data fed LSTM model to predict specific timeframes into the future.

Solar System Simulation with General Relativity

Using the rules of general relativity, I simulated our sun and planets, with their interactions on one another. The project used Python to calculate how GR affects the motion of objects and returns the next positions of those objects.
